Let's commence the description of my project

Develop a complete school management system to automate and simplify administrative, academic and communication tasks between the different stakeholders (administration, teachers, students and parents)


Project name: Madrasaty
Technologies used
• Backend: Spring Boot
• Frontend: Angular

1. Main features

• User Management:

Authentification et autorisation : Système sécurisé de connexion pour les administrateurs, enseignants, étudiants et parents, avec différents niveaux d'accès
Gestion des profils : Création, mise à jour et suppression des profils des utilisateurs

• Student management:

Registration and admission: Online registration process and admissions monitoring
Academic monitoring: Recording of grades, attendance, and student progress

• Teacher management:

Timetable: Planning and management of course schedules
Educational resources: Sharing documents and resources with students

• Course management:

Creation and management of courses: Definition of subjects, timetables and responsible teachers
Results tracking: Recording and analyzing exam and assignment results

• Communication:

Notifications: Sending notifications and announcements to students, parents and teachers
Messaging: Internal messaging system to facilitate communication between users

• Administration:

Financial management: Tracking tuition payments, managing scholarships and grants
Reports and Statistics: Generating reports on academic performance, attendance and finances

• Parent portal:

Access to information: Parents can view their children's results, attendance and progress
Interaction with the school: Possibility of contacting teachers and administration

2. Architecture

2.1. Backend (Spring Boot)

• RESTful API: Provides secure endpoints for all system operations

• Data Management: Using JPA/Hibernate to interact with the database

• Security: Implementation of JWT for authentication and authorization

2.2. Frontend (Angular)

• Responsive UI: Using Angular components to create a dynamic and responsive user interface

• Communication with the backend: Using Angular services to consume RESTful APIs provided by Spring Boot

• User Experience: Intuitive and easy-to-use design for all types of users (administrators, teachers, students, parents).

3. Benefits for the school

• Increased efficiency: Automation of administrative tasks, saving time and reducing errors

• Transparency: Better visibility into academic performance and finances

• Improved communication: Facilitation of exchanges between the different stakeholders

• Online Access: Ability to access the system anytime and from anywhere, promoting flexibility

• Tracking and reporting: Advanced tracking and reporting tools for informed decision-making



This school management system aims to modernize and optimize the daily operations of the establishment, while providing a better experience for all users involved
