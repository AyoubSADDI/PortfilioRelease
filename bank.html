Let's commence the description of my project

Gestinova is a project developed for a bank in Spain, using Spring Boot and Angular technologies, and carried out by the company Algebra System. Here is a detailed description of the project:


Project name: Mortgage
Company:Algebra System

Technologies used
• Backend: Spring Boot
• Frontend: Angular

1. The project's objectives

• Customer Management: Enable the bank to effectively manage customer information, including opening accounts, updating personal information, and managing requests for banking services

• Financial Transactions: Make it easier to manage financial transactions, such as deposits, withdrawals, transfers, and bill payments

• Security: Ensure a high level of security to protect sensitive customer data and ensure secure transactions

• Reporting and Analytics: Provide reporting and analysis tools to help the bank make informed decisions based on financial data and customer trends

• User Interface: Providing an intuitive and responsive user interface using Angular, enabling a smooth user experience for both customers and bank employees

2. Main features

• Authentication and authorization: Secure login system for users, with different access levels depending on role (administrator, employee, customer)

• Customer Dashboard: Overview of accounts and transactions for customers

• Account management: Creation, modification, and deletion of bank accounts

• Transfers and payments: Interface for making internal and external transfers, as well as bill payments

• Customer support: Support module to manage customer requests and complaints

• Notifications: Notification system to inform customers and employees of important activities

3. Architecture

• Backend (Spring Boot): Management of business logic, transactions, security, and interactions with the database

• Frontend (Angular): Development of the user interface, management of user interactions, and communication with the backend via RESTful APIs

4. Advantages for the bank:

• Operational efficiency: Automation of banking processes, reduction of human errors, and improvement in the speed of services

• Customer satisfaction: Improved user experience through a modern interface and fast services

• Enhanced security: Protection of sensitive data and prevention of fraud with advanced security mechanisms



This project aims to modernize the bank's banking operations in Spain, using cutting-edge technologies to provide secure and efficient services for both customers and bank staff
